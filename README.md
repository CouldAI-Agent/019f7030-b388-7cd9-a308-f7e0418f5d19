# 二手闲置交易 App

这是一个基于 Flutter 的跨平台二手交易应用程序，支持 Android 和 iOS 端运行。应用界面设计参考了闲鱼等主流闲置交易平台，致力于提供简单、高效的发布和浏览体验。

## 主要功能

- **用户系统**：支持用户注册、登录。
- **首页探索**：提供商品信息流瀑布流展示及顶部搜索商品功能。
- **商品发布**：支持用户填写商品标题、描述、价格，以及模拟上传商品图片。
- **即时消息**：支持模拟的用户间即时聊天功能，方便买卖双方沟通交流。
- **个人中心**：查看个人资料、我发布的商品、我买到的商品及收藏等信息。

## 技术栈

- 框架: [Flutter](https://flutter.dev/)
- 语言: Dart
- UI 风格: 现代极简风格 (Material 3)

## 快速开始

1. **安装依赖**
   确保你已安装 Flutter SDK，然后在项目根目录下运行：
   ```bash
   flutter pub get
   ```

2. **运行项目**
   支持多平台运行，推荐在移动端模拟器或真机上预览：
   ```bash
   flutter run
   ```

## 平台支持

- Android 
- iOS 
- Web (作为备用预览支持)
- Desktop (桌面端自适应支持)
